
==================================================
                 タイピングゲーム (Win7)
==================================================

このアプリは、ローカルネットワークで友人や家族と対戦できる
タイピング練習ゲームです。楽しみながらタイピングスキルを
上達させることが目的です。

--------------------------------------------------
■ 動作環境
--------------------------------------------------
- 対応OS : Windows 7 SP1 / Windows 10 / Windows 11 (32/64bit)
- CPU    : 1GHz 以上 (Pentium 4 相当以上)
- メモリ : 最小 512MB、推奨 1GB 以上
- 解像度 : 1280 x 800 以上
- その他 : 完全オフラインで動作

--------------------------------------------------
■ インストール方法
--------------------------------------------------

[ ZIP 版 ]
 1. ダウンロードした ZIP ファイルを任意のフォルダに解凍
 2. 「Typing Game (Win7).exe」をダブルクリックして起動
 3. アンインストールは、解凍したフォルダを削除するだけでOK

[ インストーラ版 ]
 1. 配布ページからセットアップファイルをダウンロード
    例: Typing Game (Win7) Setup 3.0.0.exe
        Typing Game (Win7) Setup 3.1.0.exe
       ※ バージョンによりファイル名は異なります
 2. 実行して画面の指示に従いインストール
 3. スタートメニューまたはデスクトップのショートカットから
    「Typing Game (Win7).exe」を起動
 4. アンインストールは「アプリと機能」から行ってください

--------------------------------------------------
■ ゲームの始め方
--------------------------------------------------
 1. アプリを起動
 2. 名前を入力
 3. メニューからプレイしたいモードを選択

--------------------------------------------------
■ ゲームモード
--------------------------------------------------
 1. ホームキーならし    : ホームポジションの1文字 (制限90秒)
 2. 全キーならし        : 全キーから1文字 (制限90秒)
 3. 星降るホームキー    : ホームキーの文字列 (80文字 / 120秒)
 4. 星降る全キー        : 全キーの文字列 (80文字 / 120秒)
 5. 単語練習            : 一般的な単語
 6. 文章練習            : 短文
 7. 対戦: 進捗レース    : 相手とスピード競争
 8. 対戦: 早食い        : 制限時間内にスコア勝負
 9. 苦手キー復習        : 統計データに基づく

--------------------------------------------------
■ ローカルネットワーク対戦
--------------------------------------------------
[ ホストとしてプレイ ]
 1. メインメニューで「Host game」を選択
 2. 表示されたローカルIPアドレスを相手に伝える

[ 参加する場合 ]
 1. メニューで「Connect to host」を選択
 2. ホストのIPアドレスを入力

[ ネットワーク条件 ]
 - 全員が同じLAN内にいること
 - ポート8080をファイアウォールで開放すること

--------------------------------------------------
■ カスタマイズ方法
--------------------------------------------------

[ ZIP 版 ]
 - 単語:     resources/assets/word 内の JSON を編集
 - レイアウト: resources/assets/layouts 内の JSON を編集
 - 言語:     resources/assets/lang 内の JSON を編集

[ インストーラ版 ]
 - インストール先フォルダ (例):
   C:\Program Files\Typing Game (Win7)\resources\assets\
 - 編集内容は ZIP版と同じ (word / layouts / lang)
 - 管理者権限が必要な場合があります

--------------------------------------------------
■ FAQ
--------------------------------------------------
Q: 起動時に「PCが保護されました」と表示される
A: 「詳細情報」→「実行」をクリックしてください

Q: 起動しない、画面が真っ白
A: グラフィックドライバを更新し、管理者として実行してください

Q: 512MB のメモリで動作する？
A: 基本機能は動作しますが、快適さを考えると 1GB 以上を推奨します

Q: 対戦ができない
A: LANに正しく接続しているか、ポート8080がブロックされていないか確認してください
